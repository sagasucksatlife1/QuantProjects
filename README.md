# 🚀 Featured Project

## 🧠 GD Nexus — AI Terminal for Quantitative Research

**GD Nexus** is a local-first quantitative research platform that combines autonomous strategy generation, institutional-grade backtesting, and portfolio construction into a single research environment.

The platform operates through a council of **10 specialized LLM research agents**, each representing a different investment philosophy (trend following, mean reversion, statistical arbitrage, macro, volatility, options, and more). These agents independently propose trading strategies, which are automatically validated through a secure sandboxed backtesting engine.

### Key Features

- 🤖 Multi-Agent AI Strategy Generation
- 📈 Institutional-Grade Backtesting Engine
- 🛡️ Sandboxed Strategy Execution
- 📊 Mean-Variance Portfolio Optimization
- 🌳 Hierarchical Risk Parity (HRP)
- ⚖️ Portfolio Strategy Fusion
- 📉 Strategy vs Portfolio Comparative Testing
- 📝 Complete Strategy Archive
- 🔒 Fully Local Inference (No Cloud APIs)

Every generated strategy—successful or unsuccessful—is permanently archived with its:

- Investment thesis
- Mathematical formulation
- Generated source code
- Backtest performance
- Risk statistics
- Decision history

All inference is performed locally using **Ollama + Qwen3**, ensuring complete privacy with **zero external API dependencies**.
